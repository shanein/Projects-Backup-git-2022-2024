# Project Environment

Needed : python at least 3.9 || Working fine with 3.11.5 || ne marche pas avec 3.12 apriori

# Python VENV

1. For creating :
   python -m venv env_name

2. For using :
   source env_name/bin/activate

3. Install dependencies
   pip install -r requirements.txt

4. Run the project
   python main.py
